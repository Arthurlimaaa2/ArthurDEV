    JAVA SCRIPT DATA TYPE
string: para textos
number: para números
boolean: para verdadeiro/falso
undefined: para variáveis não inicializadas
bigint: Números muito grandes (123n).
symbol: Valor único e imutável.
    null: representa "nenhum valor"
    object: para objetos
        TypedArray: para arrays
        Function: para funções
        Date: para datas
        RegExp: para expressões regulares
        Error: para erros
        arrays: para arrays
        sets: para conjuntos
        maps: para mapas
